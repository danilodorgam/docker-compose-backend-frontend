# docker-compose-backend-frontend
Esse projeto é composto com a seguinte estrutura:
```
.
+-- README.md
+-- frontend
|   +-- nginx
|      +-- default.conf
|   +-- projeto
|      +-- README.md
|   +-- Dockerfile
+-- java
|   +-- projeto
|      +-- README.md
|   +-- Dockerfile
```
Coloque seu código front tanto do frontend quanto do backend java nos respectivos diretorios "projeto"
